Speedy Spot

A mobile application that aims to alleviate parking congestion at Stony Brook University.

Description

Speedy Spot is a Android and iOS app that helps alleviate parking congestion at Stony Brook University. It is a crowdsourcing application, by which faculty, and students can input their current parking location. With enough users, we could estimate the availability of each lot and how occupied a particular lot may be.
    Features
  • Create an account
  • View lots by filter (All lots, Faculty, etc.)
  • View a specific lot's availability
  • Check in to a lot (Regular or Handicap spot)
  • Check out of a lot
  • Update account information
  • Apple maps and Google maps redirect to get directions to a specific lot

Frontend: Javascript (React Native)
Backend: AWS RDS (Relational Database Service)
Authors: Kushal Delhiwala, Austin Biegler, and Manpreet Singh of Stony Brook University.
Soon to be deployed on Google Play Store
