Metro Map Maker

A metro map creator and editing application.

Description

Metro Map Maker is an application that allows users to create and edit metropolitan subway and train maps. This is a desktop application that was coded primarily in Java.
    Features
  • Create train lines
  • Create station objects
  • Add stations to an existing train line
  • Edit postition of stations and lines via drag and drop
  • Add images to canvas
  • Customize appearance of lines and stations
  • Save current state of canvas for later continuation (via JSON files)
  • Hover Tooltips (stored in XML files)

This project was an individual academic assignment.
