Owlz

A Night Club user and establishment application.

Description

Owlz is an application that allows users to keep up with relevant events and for establishments to post events.
    Features
  • Buy tickets to events
  • View events (filter through events)
  • Post events as an establishment
  • Create an account (User or Establishment)
  • Edit user information
  • View purchase history
  • Manage employees

This project was a team academic assignment.
