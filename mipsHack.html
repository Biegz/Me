MipsHack

A 2D role playing dungeon game.

Description

Inspired by the 1987 ASCII game NetHack, MipsHack is a 2D role playing dungeon game that was completed in the fall of 2018. The user plays as the heroic knight, SirCodesALot, where he must search the dungeon for riches. On his quest he will run into minions and even bosses. Once he collects at least 3 coins of treasure he can then escape the dungeon.

The game is based primarily on two objects, the map and the player.

The Map - Upon running the program within the Mars IDE, the game will load based off of a specified text file. A normal MipsHack text file contains the map itself as ASCII characters where each character has its own representation.

    Map
  • @ = SirCodesAlot
  • . = empty floo
  • # = a wall
  • / = a door
  • > = a dungeon exit
  • $ = a single coin
  • * = a single gem, worth 5 coins
  • m = a minion monster
  • B = a boss monster
  • ? = inaccessible area

The Player - The players' state loads similar to the way the map loads. The player has a current location (cell row, cell column), health, and number of coins. These values are stored in the same text files as mentioned above.

When the game is initially started, the user cannot see where any loot, minions or bosses may be. Only when the user moves within one cell of an object will that object become visible. The game uses WASD to move SirCodesALot Up, Left, Down, and Right. The map refreshes every time the user presses a key.

This project was an individual academic assignment.
