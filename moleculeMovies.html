Molecule Movies

Database and website to browse movies.

Description

Molecule Movies is a mock database and website that allows users to browse and view information regarding movies.
    Features
  • View 'Now Playing' movies.
  • View movies A-Z
  • View actors and actresses A-Z
  • View movies by highest rating
  • View where a movie is 'playing'
  • View more information about a movie
  • Movie trailers integrated on movie pages
  • View more information about an actor

This project was a team academic assignment.
